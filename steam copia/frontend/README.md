# Frontend (Flutter)

## Requisitos

- Flutter SDK

## Configuración

La app usa `API_BASE_URL` como define de compilación.

Ejemplo:

- `flutter run --dart-define=API_BASE_URL=http://localhost:8080`

## Pantallas (MVP)

- Login / Registro
- Tienda (lista + búsqueda)
- Detalle de juego (comprar)
- Biblioteca
